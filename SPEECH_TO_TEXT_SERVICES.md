# 语音转文字服务推荐

## 🌏 国内可用的服务

### 1. **DMX API** ⭐⭐⭐⭐⭐ (推荐)
- **网站**: https://www.dmxapi.cn
- **特点**: 
  - 支持 OpenAI Whisper API 格式
  - 国内访问稳定
  - 价格实惠
  - 支持中英文
- **配置**:
  ```
  Base URL: https://www.dmxapi.cn/v1
  API Key: 你的密钥
  ```

### 2. **API2D**
- **网站**: https://api2d.com
- **特点**:
  - OpenAI API 中转服务
  - 支持 Whisper
  - 国内访问快
- **配置**:
  ```
  Base URL: https://api2d.com/v1
  API Key: fk-xxxxx
  ```

### 3. **CloseAI**
- **网站**: https://www.closeai-asia.com
- **特点**:
  - 亚洲节点
  - 支持 Whisper
  - 稳定性好

### 4. **阿里云 ASR** (语音识别)
- **网站**: https://www.aliyun.com/product/nls
- **特点**:
  - 阿里云官方服务
  - 中文识别准确
  - 需要单独集成
- **注意**: 需要修改代码适配阿里云 API

### 5. **腾讯云 ASR**
- **网站**: https://cloud.tencent.com/product/asr
- **特点**:
  - 腾讯云官方
  - 中文效果好
  - 需要单独集成

### 6. **讯飞开放平台**
- **网站**: https://www.xfyun.cn
- **特点**:
  - 国内领先的语音识别
  - 中文准确率高
  - 需要单独集成

## 🔧 如何配置

### 方法1: 使用兼容 OpenAI 的服务 (推荐)

1. 打开应用的 **Settings** 标签
2. 找到 **API Base URL** 和 **API Key**
3. 填入你选择的服务商信息
4. 保存配置

**示例 (DMX API)**:
```
API Base URL: https://www.dmxapi.cn/v1
API Key: sk-rTMJ9v5Ygs2V5lLVgB5n5fd5fH8dIuUEqApECBb746qPnqAV
```

### 方法2: 本地 Whisper (未来支持)

使用 whisper.cpp 在本地运行，完全免费但需要：
- 下载 Whisper 模型
- 较好的 CPU/GPU
- 处理速度较慢

## ⚠️ 注意事项

1. **文件大小限制**: 
   - Whisper API 限制 25MB
   - 长视频建议先压缩或剪辑

2. **处理时间**:
   - 音频: 通常 < 文件时长
   - 视频: 需要先提取音频，稍慢

3. **准确率**:
   - 清晰录音: 95%+
   - 有噪音: 80-90%
   - 多人对话: 可能需要后期校对

4. **语言支持**:
   - 中文: ✅ 支持
   - 英文: ✅ 支持
   - 其他语言: 取决于服务商

5. **成本控制**:
   - 1分钟音频 ≈ ¥0.006 (DMX API)
   - 10分钟视频 ≈ ¥0.06
   - 建议设置月度预算

## 🔐 隐私说明

- **API 服务**: 音频会上传到服务商服务器
- **本地 Whisper**: 完全本地处理，不上传
- **建议**: 敏感内容使用本地方案

## 📞 获取帮助

1. **API 配置问题**: 检查 Base URL 和 API Key
2. **转写失败**: 查看控制台错误信息
3. **文件太大**: 压缩或分段处理
4. **准确率低**: 尝试提高录音质量
